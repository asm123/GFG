Microsoft Internship Test Questions

Written by Abhijit Annaldas on Thursday, February 24, 2011 / Labels: CodeProject, Microsoft Internship, Microsoft Internship Test Questions, MSIDC Microsoft, Test Questions

Hi friends...

Today I faced Microsoft Internship Test at Microsoft India Development Center (MS IDC), Hyderabad. It was a really amazing experience. I was asked 5 questions in the written test. And I was among the selected 16 students.

Written test questions:
1] Was given a code to delete a node from a singly linked list. I don't remember the code. But there were 3 to 4 bugs. If you concentrate, you can find it out.

2] Given an array A of size N, implement two stacks starting at opposite ends of the array. Implement Push, Pop, IsEmpty and IsFull functions.

3] You are given a sorted array, possibly with duplicates, you need to return the number of array elements after eliminating all the duplicates from it. Eg: Consider array {1,2,3,3,4,5,6,6,6,7,8,9,9} has a size 13.
You should remove duplicates - {1,2,3,4,5,6,7,8,9} and return the size 9.
int compactArray(int *p, int size)
{

}

4] Design an alarm clock for a blind person. Explain what features will you include.

5] Give a test cases to test the user's login using a fingerprint scanner.

After the written test, there was another written test. It included two problems to be solved in any language C, C++, java, etc...

1] Eliminate all duplicate characters from a string array. For Eg: if you pass "BANANAS" to the function, it should return "BANS". The repeated characters must be removed from the array itself without using another temporary array. It should not be a n square solution.

2] Write a function to check if a given binary tree is balanced. Calculate space and time complexity. Write suitable test cases for the same.
